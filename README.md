
# ESP32 IoT Weather Station

An embedded systems project using ESP32, DHT22 sensor, and OLED display.

## Features
- Real-time temperature monitoring
- Humidity monitoring
- OLED live display
- Serial monitor output
- Wi-Fi enabled setup
- Wokwi simulation compatible

## Components Used
- ESP32 DevKit V1
- DHT22 Temperature & Humidity Sensor
- SSD1306 OLED Display

## Folder Structure
ESP32_IoT_Weather_Station/
│
├── weather_station.ino
├── diagram.json
└── README.md

## Wokwi Simulation
1. Open Wokwi
2. Create a new ESP32 project
3. Replace the generated files with:
   - weather_station.ino
   - diagram.json
4. Start simulation

Official Website:
https://wokwi.com
